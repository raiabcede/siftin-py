Frequently Asked Questions

Siftin is a lead generation tool that helps you capture and manage LinkedIn leads. It allows you to search LinkedIn, define AI criteria to filter leads, and export them to your CRM or as CSV files.
The AI criteria feature allows you to describe in natural language what kind of leads you're looking for. For example, "US-based SDR managers at B2B SaaS companies with 11-200 employees" will help filter leads that match your criteria.
Yes! Siftin supports exporting leads to popular CRMs like HubSpot, Salesforce, and Pipedrive. You can also export leads as CSV files for use in other systems.
Siftin uses deduplication rules to identify and prevent duplicate leads. You can configure this in Settings, choosing to dedupe by email address, LinkedIn URL, or name + company combination.
Siftin supports LinkedIn search results URLs that contain "/search/results/people". Profile URLs like /me or /in/ are not supported. Make sure you're using a search results URL from LinkedIn's people search.
This is a UI prototype demonstration. While the interface is fully functional, actual LinkedIn scraping and data collection would require proper implementation and compliance with LinkedIn's Terms of Service and applicable regulations.
